# Case Ladder Mobile App

This folder is the phone-friendly version of the Case Ladder MCQ tool.

## Best way to use

Host this whole folder on a static host:

- Netlify
- Vercel
- Cloudflare Pages
- GitHub Pages

Then open the hosted link on iPhone Safari or Android Chrome.

## iPhone

1. Open the hosted link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Open Case Ladder from the Home Screen.

## Android

1. Open the hosted link in Chrome.
2. Tap the menu.
3. Tap Add to Home screen or Install app.

## Why this is better than sending HTML

Phone browsers are unreliable with loose local HTML files. A hosted PWA avoids broken local file paths, preview wrappers, blocked storage, and missing assets.

## Offline behavior

After first load, the app shell is cached for offline use. Progress is saved in the browser on that device.
